# Thumbnail Designer — AI Music Lab

## Контекст канала
- Канал: AI Music Lab (русскоязычная аудитория)
- Формат: talking-head видео + музыкальный трек
- Инструменты: Canva (дизайн) + AI-генератор изображений (фон/визуал)
- Стиль: спокойный, наблюдательный, личный

---

## ПРИНЦИПЫ ВЫСОКОГО CTR (по практике топ-каналов)
- **Контраст цвета:** насыщенный акцент на тёмном/нейтральном фоне — жёлтый на тёмно-синем, красный на чёрном/циан, синий на оранжевом. Комплементарные пары всегда выигрывают у низкоконтрастных сочетаний.
- **Правило третей:** холст делится на сетку 3×3, главный объект/лицо — в одной из четырёх точек пересечения линий, не по центру кадра.
- **Эмоция на лице:** сильная, читаемая эмоция (удивление/шок/задумчивость) поднимает CTR заметно сильнее, чем текстовая обложка без лица — если в кадре есть лицо, эмоция важнее деталей фона.
- **Текст:** максимум 3-5 слов, жирный шрифт без засечек, обводка или тень для читаемости. Обложка должна читаться в размере мобильного превью (примерно 120px по ширине) — если не читается мелким, текст слишком длинный или мелкий.
- **Простота:** не больше 3 смысловых элементов в кадре (фон + герой/объект + текст). Перегруженная обложка проигрывает при просмотре в ленте.

## ПРАВИЛО №0 — Никогда не рисуй интерфейс буквально
Для обучающих/технических тем («как сделать в Suno», «два поля → одна песня», разбор функции) — САМАЯ частая ошибка: попросить AI-генератор нарисовать скриншоты интерфейса, окошки приложения, панели с текстом внутри. Результат всегда плоский и дешёвый: генератор изображений не умеет рисовать мелкий читаемый UI-текст, получается мыльная имитация интерфейса вместо обложки.

**Вместо этого — переведи суть урока в одну сильную визуальную метафору**, как кинопостер:
- Тема «как структурировать трек по актам» → режиссёр в кресле, хлопушка, кинолента (метафора монтажа/режиссуры)
- Тема «два поля → одна песня» → например: два потока/дороги/луча сливаются в один мощный поток света; или руки соединяют два пазла; конкретный предмет-метафора, а не скриншот формы ввода
- Тема «7 ошибок в промптах» → разбитое зеркало, красные крестики на объектах-метафорах ошибки, взрыв листа бумаги — НЕ скриншот текстового поля с зачёркнутым текстом

**ЗАПРЕЩЕНО в промпте для AI-генератора:**
- screenshot, UI, interface, app window, text field, dialog box, form, browser window
- несколько мелких окон/панелей рядом друг с другом
- диаграммы со стрелками между текстовыми блоками

**Разрешено и приветствуется:** один крупный визуальный образ-метафора + сильный свет/контраст + герой в кадре (силуэт/фигура), как в кино-постере — интерфейс можно упомянуть только словом в заголовочном тексте («SUNO», «Custom Mode»), но не рисовать как картинку интерфейса.

## ПРАВИЛО №1 — Принцип комплементарности
Текст на миниатюре НЕ повторяет название видео.
- Название: «5 треков которые я сделал за ночь»
- Миниатюра: «03:00. Не сплю.»

**Формулы текста на обложке:**
- Время + состояние: `03:00. Не сплю.`
- Число + факт: `47 попыток`
- Незаконченное действие: `Я просто нажал...`
- Провокация: `Это не музыкант`
- Вопрос без ответа: `Почему все плачут?`

**ЗАПРЕТ:** Никаких эмоциональных ярлыков («грустная», «эпичная») и жанровых описаний («неоклассика», «EDM»).

## ФАЗА 1: Определи тип миниатюры

### Тип A — Talking Head (Александр в кадре)
Используй когда: видео начинается с личной истории, есть сильная эмоция или реакция.

```
Композиция:
- Александр занимает 40-50% кадра (левая или правая треть)
- Взгляд направлен на зрителя или в сторону текста
- Сильное выражение лица: удивление / задумчивость / спокойная уверенность
- Текст на противоположной стороне — крупно, 2-4 слова
- Фон: размытый интерьер или градиент
```

### Тип B — Атмосферный визуал (без лица)
Используй когда: инструментальный трек, медитативная тема, ночная/дождливая атмосфера.

```
Композиция:
- Один сильный визуальный образ (окно с дождём, огонь свечи, ночной город)
- Текст поверх — крупно, с обводкой или на тёмной полосе
- Цвет: тёмная палитра с одним акцентом
```

### Тип C — Split / Контраст
Используй когда: есть сравнение, до/после, противопоставление.

```
Композиция:
- Левая половина: «до» (тёмная, десатурированная)
- Правая половина: «после» (яркая, насыщенная)
- Разделитель: вертикальная линия или градиент
- Одно слово на каждой стороне
```

### Тип D — Концептуальная метафора (обучающие/технические темы)
Используй когда: тема — урок, туториал, разбор функции или шагов («как сделать X», «два поля → песня», «N ошибок»). НЕ используй буквальный скриншот интерфейса (см. ПРАВИЛО №0).

```
Композиция:
- Один сильный символический образ, переводящий суть урока в метафору
  (режиссёр в кресле = структура/монтаж, два потока света сливаются в один = объединение полей,
  разбитое зеркало = ошибки, ключ в замке = разблокировка функции)
- Драматичный боковой/контровой свет, кинематографичный контраст
- Крупный стилизованный заголовок (текстура/тень/градиент на буквах), 2-4 слова
- Опционально — силуэт человека-персонажа как якорь кадра
- Никаких экранов, окон, панелей, полей ввода
```

## ФАЗА 3: Сборка в Canva

### Правила текста
- Максимум 4-5 слов
- Размер: 72-120pt для главного текста
- Всегда добавляй обводку (4-6px чёрная для светлого текста)
- Или: белый текст на полупрозрачной тёмной полосе

### Цветовые схемы для AI Music Lab

| Настроение | Фон | Текст | Акцент |
|-----------|-----|-------|--------|
| Ночь/грусть | #0D1B2A (тёмно-синий) | #FFFFFF | #C9A84C (золото) |
| Энергия/EDM | #0A0A0A (чёрный) | #00D4FF (циан) | #FF006E (маджента) |
| Тепло/душа | #1A0A00 (тёмно-коричневый) | #FFD700 (золото) | #FF6B35 (оранжевый) |
| Эпик/хор | #1A0000 (тёмно-красный) | #FFFFFF | #C9A84C (золото) |
| Спокойствие | #F5F0E8 (бежевый) | #2C2C2C (тёмный) | #8B7355 (коричневый) |

### Чеклист перед публикацией
- [ ] При размере 120x68px (превью) текст читается
- [ ] За 3 секунды понятно о чём видео
- [ ] Текст НЕ повторяет название
- [ ] Максимум 3 элемента (фон + лицо/визуал + текст)
- [ ] Проверено на тёмном и светлом экране
